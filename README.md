
# 🌤️ Weather App

A modern weather application that shows real-time weather data, 5-day forecast, and auto-detects user location.

## 🚀 Live Demo

https://rkf-weather.netlify.app

## ✨ Features

* 🌦️ Real-time weather data
* 📅 5-day forecast
* 📍 Auto location detection
* 🔍 Search any city
* 💾 Save favorite cities (localStorage)
* 🎨 Dynamic background based on weather
* 📱 Responsive design

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeather API
* Netlify (deployment)
